Top 3 Projects I’ve Built in 2025

This year has been a huge leap in my development journey. I’ve designed, coded, and deployed some of my most meaningful and challenging projects yet. Here are my top three:

1. MyAfya Hub Clone
A clean, responsive web version of the MyAfya Hub app, featuring static screens, mock dashboards, and mobile-first design. I recreated complex UI interactions using only CSS animations and layout tricks.

2. Portfolio Version 3.0
This very portfolio site! It’s my best iteration so far — responsive, lightweight, and JavaScript-free. It includes a blog, testimonials carousel, contact form, and dynamic UI elements — all with HTML and CSS.

3. CSS-Only Image Gallery
A creative project where I built a responsive, filterable gallery using no JavaScript. With `:checked` toggles, pure CSS grid, and transitions, I proved what CSS can do on its own.

These projects pushed me technically and creatively — and they continue to inspire my future builds.
